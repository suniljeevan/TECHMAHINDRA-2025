PROJECT GUIDELINES:

1. Find the directory named scms in the scholarship management branch.

2. Download the file. 

3. Import it as a maven project.

4. Change the username and password in the application.properties file

5. create a database in mysql and name it as sscms.

6. Run the project as springboot app.

7. For admin login use:
		username: admin
		password: admin123
8. For student login use:
		username: student
		password: student123
